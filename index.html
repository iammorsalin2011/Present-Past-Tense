Present Indefinite Tense

Structure:

  • Subject + verb + extension

Present Continuous Tense

Structure:

  • Subject + am/is/are + verb + ing + extension

Present Perfect Tense

Structure:

  • Subject + have/has + v3 + extension

Present Perfect Continuous tense

Structure:

  • Subject + have been/has been + verb + ing + -----
  • Subject + have been/has been + verb + ing + since/for + -----

Past Indefinite Tense

Structure:

  • Subject + v2 + extension

Past Continuous Tense

Structure:

  • Subject + was/were + verb + ing + extension

Present Perfect Tense

Structure:

  • Subject + had + v3 + before + Past Indefinite
  • Past Indefinite + after + Subject + had + v3

Present Perfect Continuous Tense

Structure:

  • Subject + had been + verb + ing + since/for ----- + before + Past Indefinite
